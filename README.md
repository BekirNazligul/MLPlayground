# ML Playground

This project is a collection of ML experiments, implementation from scratch and various vibe coded things.

Intended for learning ML from first principals and torturing my GPU.

## Features

- Everything is aggressively annotated with [jaxtyping](https://github.com/patrick-kidger/jaxtyping) library, because I always have troubles with tensor dimensions.
- Transformer + GPT implementation in PyTorch, adapted from [NanoGPT](https://github.com/karpathy/nanoGPT/). 100% human written. 
- Vibe coded simple training loop.
